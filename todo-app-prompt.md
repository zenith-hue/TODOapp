# TO DO 앱 제작 프롬프트

순수 HTML/CSS/JS(외부 라이브러리·프레임워크 없이)로 동작하는 단일 페이지 TO DO 앱을 만들어줘.

## 기능 요구사항
- 화면에 할 일을 입력할 수 있는 텍스트 입력창과 "추가" 버튼이 있어야 한다.
- 입력창에서 Enter 키를 눌러도 할 일이 추가되어야 한다.
- 입력값이 비어있으면 추가되지 않아야 한다.
- 추가된 할 일은 화면의 목록(리스트)에 즉시 나타나야 한다.
- 각 할 일 항목에는 체크박스가 있어, 체크하면 완료 상태(예: 취소선 스타일)로 표시되고 다시 해제할 수 있어야 한다.

## 파일 구성
- `index.html`: 입력창, 추가 버튼, 할 일 목록 컨테이너 마크업
- `style.css`: 깔끔하고 간단한 레이아웃/스타일, 완료 항목 취소선 스타일
- `script.js`: 할 일 추가, 목록 렌더링, 완료 토글 로직
