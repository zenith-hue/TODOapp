# 구글 시트 + Apps Script 배포 가이드

## 1. 시트 준비
1. 새 구글 시트를 만듭니다.
2. 1행(헤더)에 `id`, `text`, `completed` 세 컬럼을 입력합니다.

## 2. Apps Script 연결
1. 시트 메뉴에서 `확장 프로그램 > Apps Script`를 클릭합니다.
2. 열린 편집기의 `Code.gs` 내용을 모두 지우고, 이 폴더의 [`Code.gs`](./Code.gs) 내용을 그대로 붙여넣습니다.
3. 저장합니다.

## 3. 웹 앱으로 배포
1. 편집기 우측 상단 `배포 > 새 배포`를 클릭합니다.
2. 유형 선택에서 `웹 앱`을 선택합니다.
3. "다음 사용자 인증 정보로 실행": `나`
4. "액세스 권한이 있는 사용자": `모든 사용자` (Anyone) — 개인용 앱이며 시트/스크립트 URL 자체를 비밀로 취급합니다.
5. `배포`를 클릭하고 구글 계정 권한 승인을 완료합니다.
6. 배포 후 발급되는 **웹 앱 URL**을 복사합니다.

## 4. 앱에 연결
`script.js` 상단의 `SHEET_API_URL` 상수에 3-6에서 복사한 URL을 붙여넣습니다.

```js
export const SHEET_API_URL = '여기에_배포된_웹_앱_URL';
```

## 5. 재배포 시 주의
`Code.gs`를 수정한 뒤에는 `배포 > 배포 관리`에서 기존 배포를 `편집`하고 버전을 `새 버전`으로 올려야 변경 사항이 실제 URL에 반영됩니다.
